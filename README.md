# EnqbatorReactTakeHome
Take home project for enqbator


Here is the front-end clone (with minor improvements compared to the original).
I could not get some of the logos as they were not public. So i improvised and added a simple placeholder for them. 

Some of the design choices, I went ahead and revamped them and made small improvements to them (for readability and style choices). Keep in mind all of this is just nothing more than a static frontend for the take home project.

# To View the frontend page:

Pull the clone the repo
cd into oakGovClone/
```
$ cd oakGovClone
```

Run the frontend:
```
$ npm run dev
```
